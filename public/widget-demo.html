Installation Instructions

Include the widget script at the end of your HTML body:

<script src="https://your-domain.com/casino-chat-widget.js"></script>

Initialize the widget:

<script>
  document.addEventListener('DOMContentLoaded', function() {
    // Initialize with default container
    window.CasinoChatWidget.init();
    
    // Or specify a custom container ID
    // window.CasinoChatWidget.init('my-custom-container');
  });
</script>

API Reference

  • init(containerId): Initialize the widget with an optional custom container ID
  • open(): Open the chat window
  • close(): Close the chat window
